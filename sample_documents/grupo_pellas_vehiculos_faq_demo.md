# FAQ demo: Vehiculos Grupo Pellas

> Documento ficticio para pruebas locales del chatbot y RAG. La informacion de
> modelos, precios, horarios, politicas y servicios es inventada para
> capacitacion. No debe usarse como informacion comercial real de Grupo Pellas.

## Reglas de respuesta

- Responder solo con informacion contenida en estos documentos demo.
- Aclarar que precios, promociones y requisitos son referenciales de demostracion.
- No confirmar inventario real.
- No aprobar credito.
- No prometer descuentos no descritos.
- Si el cliente quiere comprar, reservar, financiar o negociar, sugerir contacto con asesor.

## Preguntas frecuentes generales

### Que marcas o tipos de vehiculos cubre esta demo?

Esta demo cubre tres lineas ficticias: motocicletas urbanas y de trabajo,
vehiculos livianos familiares, y vehiculos comerciales para negocio.

### Puedo cotizar desde el bot?

El bot puede explicar precios de referencia demo y requisitos generales. Para
una cotizacion formal, reserva, disponibilidad real o negociacion, debe escalar
a un asesor humano.

### Que documentos pide la demo para comprar?

Para compra de contado: identificacion vigente, datos de contacto y direccion.
Para financiamiento demo: identificacion vigente, comprobante de ingresos,
referencias personales, prima inicial y autorizacion para revision crediticia.

### El bot puede aprobar credito?

No. El bot no aprueba creditos. Solo explica requisitos de demostracion y debe
remitir al usuario a un asesor para revision formal.

## Financiamiento demo

### Prima inicial

La prima inicial referencial de demostracion puede iniciar desde el 15% del
precio del vehiculo, sujeto a evaluacion. El bot no debe prometer aprobacion.

### Plazos

Los plazos de demostracion pueden ir de 12 a 60 meses segun perfil del cliente,
tipo de vehiculo y condiciones del plan.

### Requisitos de financiamiento

- Identificacion vigente.
- Comprobante de ingresos.
- Recibo de servicio o direccion verificable.
- Referencias personales.
- Prima inicial.
- Evaluacion crediticia.

## Taller y mantenimiento demo

### Primer mantenimiento

Para motocicletas demo, el primer mantenimiento sugerido es a los 500 km. Para
autos demo, el primer mantenimiento sugerido es a los 1,000 km o primer mes,
lo que ocurra primero.

### Garantia

La garantia demo para motos es de 12 meses o 12,000 km. La garantia demo para
autos es de 36 meses o 100,000 km. Las garantias pueden perder cobertura por
uso indebido, modificaciones no autorizadas o falta de mantenimientos.

### Repuestos

La demo indica disponibilidad referencial de repuestos de alta rotacion:
aceites, filtros, bujias, fricciones, llantas, baterias y accesorios basicos.

## Escalamiento a asesor

El bot debe sugerir contacto con asesor cuando el usuario:

- Pida precio final o descuento.
- Quiera reservar una unidad.
- Pida disponibilidad real de color o modelo.
- Solicite aprobacion de credito.
- Quiera entregar un vehiculo usado como parte de pago.
- Reporte un problema legal, garantia compleja o reclamo formal.

Datos minimos para seguimiento comercial:

- Nombre completo.
- Telefono.
- Ciudad.
- Modelo de interes.
- Compra de contado o financiamiento.
